## 基础环境

### 安装Python和虚拟环境

确保你已经安装了Python 3.8或更高版本。你可以从[Python官方](https://www.python.org/downloads/)网站下载并安装。

1.创建虚拟环境

```shell
python -m venv .venv
```

2.激活虚拟环境

```shell
  # Windows:
  .venv\Scripts\activate
  # macOS/Linux:
  source .venv/bin/activate
```

3.安装Python依赖

```shell
cd example
pip install -r requirements.txt
# or 
pip install package-name
```
